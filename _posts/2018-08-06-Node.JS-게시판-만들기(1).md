Node.js 란?
=============

Node.js 는 Javascript 로 서버 사이드를 개발 할 수 있는 플랫폼입니다.

웹 어플리케이션 개발시 클라이언트/서버 사이드를 단일 언어로 개발 할 수 있으며, Non-Blocking I/O, Single Thread 등의 장점이 있습니다.

바로 본론으로 들어가서, Node.js 기반 게시판을 만들어 보도록 하겠습니다.
